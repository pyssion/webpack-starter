# webpack-starter

Pyssion's webpack starter kit (with React ecosystem).


## features

- [ ] React with ES6 and JSX
- [ ] React-Router with code splitting
- [ ] Redux and reselect
- [ ] Rx.js for side effects
- [ ] future ready CSS
- [ ] configurable eslint
- [ ] webpack based with HMR
